📋 SETUP (3 STEPS - 5 MINUTES)

  1. Get API Key
     → Visit: https://www.firecrawl.dev/
     → Sign up → Copy key

  2. Set Environment
     $ export FIRECRAWL_API_KEY="your_key_here"

  3. Run
     $ source .venv/bin/activate
     $ python scraper_enhanced.py    # Scrape data
     $ streamlit run app_comprehensive.py  # Run app

❓ TROUBLESHOOTING

  Problem: "No data loaded"
  → Run: python scraper_enhanced.py

  Problem: "Streamlit won't start"
  → Run: uv pip install streamlit plotly pandas numpy firecrawl-py

  Problem: "API key not working"
  → Check: echo $FIRECRAWL_API_KEY
  → Reset: export FIRECRAWL_API_KEY="your_actual_key"

  Problem: "Scraper fails"
  → May hit rate limits - run again later
  → Check key at: https://www.firecrawl.dev/
