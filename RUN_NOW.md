# 🚀 Run Master Brewer Now!

## ⚡ 3-Minute Quick Start

### Step 1: Install Dependencies
```bash
cd /Users/ankur/streamlit_beer_ana
pip install -r requirements.txt
```

### Step 2: Run the App
```bash
streamlit run app.py
```

### Step 3: Open in Browser
The app will automatically open at `http://localhost:8501` ✅

## 🎯 First Things to Try

### Try #1: Quick Hop Finder (1 min)
1. Click **🎯 Quick Hop Finder** in sidebar
2. Set: IBU = 60, ABV = 6.5, Style = "IPA"
3. Click "Find Hops" button
4. See top 5 compatible hops with flavor profiles!

### Try #2: Explore a Beer Style (2 min)
1. Click **📊 Beer Style Explorer**
2. Select "INDIA PALE ALE (IPA)" category
3. Choose "American IPA" style
4. See official ranges + recommended hops

### Try #3: View Detailed Hop Card (2 min)
1. Click **🌿 Hop Cards & Details**
2. Select "Citra" from dropdown
3. See complete info:
   - Alpha acids
   - Flavor profile
   - Best beer styles
   - Compatible pairings
   - Dosage guidelines

### Try #4: Discover Hop Pairings (2 min)
1. Click **🤝 Hop Pairings**
2. Select "Citra" as primary hop
3. See compatible pairings
4. View combined flavor wheel

### Try #5: Calculate Dosages (2 min)
1. Click **💧 Dosage Guidelines**
2. Select "Citra" hop
3. See recommended amounts:
   - Bittering stage
   - Aroma stage
   - Dry hop
4. Use IBU calculator

### Try #6: See Flavor Combinations (2 min)
1. Click **🔬 Flavor Analysis**
2. Select 3 hops: Citra, Mosaic, Nelson Sauvin
3. See interactive flavor wheel
4. View flavor category distribution

### Try #7: Check Hop Popularity (2 min)
1. Click **📋 Trends & Popularity**
2. See ranking of most versatile hops
3. View popularity scores
4. Check origin distribution

### Try #8: Build Your First Recipe (3 min)
1. Click **🧪 Recipe Builder**
2. Select style: "American IPA"
3. Adjust ABV (6-7%), IBU (60-70)
4. Pick hops:
   - Bittering: Simcoe
   - Aroma: Citra, Mosaic
   - Dry: Nelson Sauvin
5. Click "Generate Recipe Summary"
6. See complete recipe!

## 🔧 Troubleshooting

### App won't start?
```bash
# Make sure dependencies are installed
pip install --upgrade streamlit pandas numpy plotly

# Try running again
streamlit run app.py
```

### Port 8501 already in use?
```bash
# Use a different port
streamlit run app.py --server.port 8502
```

### Data files not found?
Ensure you have:
- `data/bjcp-style-data/csv/beer-vital-statistics-2008.csv`
- Your data directory structure is intact

Check these files:
